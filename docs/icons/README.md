# Icônes PWA

Pour que la PWA fonctionne correctement, vous devez ajouter deux icônes :

- **icon-192.png** : 192x192 pixels
- **icon-512.png** : 512x512 pixels

## Comment créer vos icônes

1. Créez un logo carré (512x512px minimum)
2. Utilisez un outil comme :
   - [Favicon.io](https://favicon.io/)
   - [RealFaviconGenerator](https://realfavicongenerator.net/)
   - Photoshop / GIMP / Canva

3. Exportez en PNG avec fond transparent ou couleur unie
4. Nommez les fichiers `icon-192.png` et `icon-512.png`
5. Placez-les dans ce dossier `docs/icons/`

## Icônes temporaires

Les fichiers actuels sont des placeholders. Remplacez-les par vos vraies icônes avant le déploiement.
